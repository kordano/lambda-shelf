# lambda-shelf

A basic stuff collector using David Nolens React interface [om](https://github.com/swannodette/om "om").

## Usage

Build clojurescript file

```
lein cljsbuild once
```

Start backend servers
```
lein run
```

Visit [localhost:8080](http://localhost:8080 "Lambda Shelf")
